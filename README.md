FinTechUI — Personal Finance Dashboard

# Tech Stack used:
```
- Next.js
- TailwindCSS
- Shadcn
- Recharts
```

# Usage
``` 
- Clone the repo
git clone https://github.com/Lunar-spec/FinTechUI.git

- Install packages
npm install
# or
yarn install
# or
pnpm install
# or
bun install

- Run the development server
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
Open http://localhost:3000 with your browser to see the result.

# Features
- Summary Cards.
- Various charts for proper visualtion of data.
- Transaction table with pagination and sorting along with category filter.
- Toggle Role.
- Toggle Theme.
- Centeralised data at `data/transactions.ts`.
- Custom tooltips and legends for better UI experience.
- State management for Theme, Role, Category filter.
